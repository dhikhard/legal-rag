# Legal RAG System
> An AI-powered legal document Q&A system built with RAG (Retrieval-Augmented Generation)

## What it does
Upload Indian legal documents (IPC sections, court judgments, contracts) and ask questions
in plain English. The system retrieves the most relevant legal passages and generates
accurate, citation-backed answers.
